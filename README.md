Student Connect – Smart Academic Collaboration Platform | React.js, Node.js, Express.js, MySQL
Developed a full-stack web application that centralizes academic communication, event management, study resources, and assignment tracking for students and faculty. Built a responsive and user-friendly interface using React.js and implemented secure Authentication and Role-Based Authorization using JWT. Developed RESTful APIs with Node.js and Express.js and designed a normalized MySQL database for efficient data storage and retrieval. Implemented key features including announcement management, event registration, study resource sharing, assignment tracking, dashboard analytics, and complete CRUD operations. Enhanced collaboration and communication through role-based dashboards, real-time updates, and a modern responsive design.
